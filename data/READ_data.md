# Data used in the study

This directory contains the segmented 2D pore-mask images used in the Pore-DCF study.

The images correspond to three different 2D rock structures analyzed in the work:

```text
patch_y3800_x3800_c0_mask.png
patch_y7600_x19000_c0_mask.png
patch_y7600_x53200_c0_mask.png
```

## Image format

Each image is a segmented RGB mask:

- green pixels represent the pore space;
- black pixels represent the solid matrix.

The Pore-DCF descriptor uses this binary pore/matrix information to compute the internal distance map, select probe-particle radii, simulate dynamic collisions inside the pore space, and extract morphological, free-path, angular entropy, and Fourier descriptors.

## How to use these images

To analyze one of the images, edit the `IMAGE_PATH` variable in `descriptor.py`.

Example:

```python
IMAGE_PATH = "data/patch_y7600_x53200_c0_mask.png"
```

Then run:

```bash
python descriptor.py
```

This generates the descriptor file:

```text
pore_dcf_descriptor.csv
```

To generate the interpretability figures for the same structure, edit the following variables in `interpretability.py`:

```python
IMAGE_PATH = "data/patch_y7600_x53200_c0_mask.png"
CSV_PATH = "pore_dcf_descriptor.csv"
```

Then run:

```bash
python interpretability.py
```

## Notes

The same image path should be used in both `descriptor.py` and `interpretability.py`.

The `CSV_PATH` in `interpretability.py` must point to the descriptor file generated by `descriptor.py`.
